text-transform : 영문 텍스트의 대소문자 변환시 사용

    선택자{
        text-transfrom : uppercase | lowercase | capitalize
    }
  1. uppercase : 모든 영문자를 다 대문자로
  2. lowercase : 모든 영문자를 다 소문자로
  3. capitalize : 영문자의 firsttext만 대문자로

text-shadow : 텍스트에 그림자 효과를 줄 때 사용

    선택자 {
        text-shadow : 가로거리(x) 세로거리(y) [빈짐정도][색상]
    }
